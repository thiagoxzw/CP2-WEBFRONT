# Kyoto Garden — Restaurante Japonês

Site institucional dinâmico e responsivo do restaurante japonês **Kyoto Garden**, desenvolvido como projeto da CP2 de Web Front-End.

## Tema
Restaurante Japonês — Kyoto Garden 京都ガーデン

## Estrutura do Projeto

```
.
├── index.html
├── pages/
│   ├── portifolio.html
│   ├── sobre.html
│   └── contato.html
└── src/
    ├── assets/
    │   ├── img/
    │   │   └── logo.png
    │   └── pdf/
    ├── css/
    │   └── estilo.css
    └── js/
        └── script.js
```

## Páginas

- **Home** — Hero principal com logo, apresentação do restaurante e cards de diferenciais.
- **Portifólio** — Cardápio completo com 6 pratos (sushi, ramen, sashimi, tempura, yakisoba, mochi) e tabela de combos promocionais.
- **Sobre** — História do Chef Hiroshi Tanaka e a filosofia do restaurante em layout de 2 colunas.
- **Contato** — Formulário de contato com nome, e-mail e mensagem.

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, Flexbox, Grid, responsividade)
- JavaScript vanilla (interatividade do menu e formulário)

## Identidade Visual

- **Logo:** criado no Claude Design (Canva)
- **Paleta:** vermelho `#c8102e`, preto `#0a0a0a`, dourado `#d4af37`, creme `#faf3e7`
- **Tipografia:** serifada (Georgia)
- **Fotos:** Unsplash (domínio público)

## Como Executar

Abra o arquivo `index.html` no navegador, ou hospede via GitHub Pages.
